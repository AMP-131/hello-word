    • Hola mundo, esta será mi primera página utilizando Bootstrap.

    Bootstrap es un tipo de framework que conjunta HTML, CSS y JavaScrip/JQuery para la creación de páginas web. Utilizando solo te tendrás que preocupar del contenido de tu página y él se encargara de ponerlo en marcha, por ejemplo para colocar, modificar, diseñar,… algún apartado de tu página web solo tienes que encontrar el contenido que necesitas copiarlo-pegarlo y modificarlo a tu gusto.

    Bootstrap no es un programa, es una “librería” donde almacenamos nuestros archivos de nuestra página web, y esta se encargará de adaptare al navegador según su tamaño.
